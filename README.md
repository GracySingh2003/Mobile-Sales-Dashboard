# Mobile-Sales-Dashboard

📊 **Project Title / Headline:**<br/>
 MOBILE SALES DASHBOARD - MOTOROLA<br/>
 This project showcases an **interactive Mobile Sales Dashboard** built using Power BI.<br/>

📝**Short Description:**<br/>
This project presents an interactive Mobile Sales Dashboard built using Power BI to analyze sales performance across different regions, brands, and time periods. The purpose of this dashboard is to transform raw sales data into meaningful insights that help businesses make data-driven decisions, identify trends, and improve overall sales strategies.<br/>

🛠️**Tech Stack:**<br/>
Power BI Desktop – Data visualization & dashboard creation.<br/>
Power Query – Data cleaning and transformation.<br/>
DAX (Data Analysis Expressions) – Calculations and measures.<br/>
Microsoft Excel / CSV – Data handling.<br/>

📂**Data Source:**<br/>
Mobile sales dataset provided as part of the **SkillCourse Power BI training program.**<br/>
Contains information such as product details, sales quantity, revenue, regions, and time-based data.<br/>

🚀**Features & Highlights**:<br/>

🔍 **Business Problem**-<br/>
Businesses often struggle to track sales performance across multiple dimensions like regions, products, and time. Raw data alone makes it difficult to identify trends, top-performing products, and revenue patterns.<br/>

🎯**Goal** - To design an intuitive and interactive dashboard that:<br/>
Provides clear insights into mobile sales performance.<br/>
Helps stakeholders identify key trends and patterns.<br/>
Supports better decision-making through visual analytics.<br/>

📈 **Key Visuals**-<br/>
KPI Cards – Total Sales, Revenue, Quantity Sold<br/>
Bar Charts – Sales by Brand / Model<br/>
Line Charts – Sales trends over time<br/>
Pie/Donut Charts – Market share distribution<br/>
Map Visuals – Region-wise sales performance<br/>
Slicers/Filters – Interactive filtering by date, region, and brand<br/>

⭐ **Key Features**-<br/>
Interactive and user-friendly dashboard<br/>
Dynamic filtering for better data exploration<br/>
Region and product-level performance insights<br/>
Clean and professional data visualization design<br/>
